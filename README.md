# CMR Manager

Bienvenu sur CRM Manager. Ce projet met à disposition des fonctionnalités basiques pour la gestion d'utilisateurs.

## Features

- Ajouter un utilisateur
- Lister les utilisateurs
- Supprimer et modifier un utilisateur
- Ajouter un contact via des flags

## How To Use It 

Lancer la commande
```bash 
go run main.go
```
